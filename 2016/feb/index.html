About

Microsoft User Group Hyderabad (MUGH) started life as DNUG (.NET Users Group) Hyderabad. We felt that various Microsoft technologies function so well in concert with each other and our earlier name did restrict our scope to just one of them. Today we have special groups branched into various fields under MUGH, namely DNUG (a dot net user group), IT Pro (a user group for it community), SQLCon (India's first SQL community), a student group. MUGH supports various other user groups from different regions.

MUGH is operated by a core group that consists of Microsoft MVP's and Community enthusiasts. Presently, MUGH's user group meetings are held every last Saturday of the month.

Find out more about events organized by MUGH on facebook, twitter and blog.
